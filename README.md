# How to play

The game ship with a total of **7 beatmaps**:

- Doodle
- Bocchi the Rock Mix
- Night of Knights
- Otsukimi
- Pokkan Color
- Pumpin' Junkies
- Tengu Man

You can tweak the scroll speed using `Left` or `Right` with the Dpad on the Beatmap selection screen.

Just pick your map and start playing with those buttons:

| Lane | Button |
|-|-|
| 1 | Left trigger |
| 2 | D-pad left |
| 3 | A |
| 4 | Right trigger |

You can go back to the beatmap selection by pressing Start.

# FAQ

> *Why no beatmap editor?*
- A beatmap editor could be nice, but it'll require a song to play on. Sadly, there's no intuitive way of importing song unless using complex methods.

> *Why not downloading maps from osu server?*
- A N64 don't have any internet connexion on its own. If using a custom cartridge, maybe some advanced build could allow such, but this wasn't the main scope.

> *I don't understand the project, is it a Super Mario 64 Rom Hack?. How was it made?*
- No its not! It's like a complete new game made especially for the N64 with its own engine using the editor Pyrite64, which works with Libdragon and Tiny64.
